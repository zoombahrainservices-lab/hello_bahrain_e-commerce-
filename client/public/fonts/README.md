# BRHendrix Font Files

Place your BRHendrix font files in this directory:

## Required Font Files:

1. **BRHendrix-Semibold.woff2** (preferred format)
2. **BRHendrix-Semibold.woff** (fallback)
3. **BRHendrix-Semibold.ttf** (fallback)

4. **BRHendrix-Regular.woff2** (optional, for regular weight)
5. **BRHendrix-Regular.woff** (optional)
6. **BRHendrix-Regular.ttf** (optional)

7. **BRHendrix-Bold.woff2** (optional, for bold weight)
8. **BRHendrix-Bold.woff** (optional)
9. **BRHendrix-Bold.ttf** (optional)

## Font Configuration:

The project is configured to use:
- **Font Family**: BRHendrix
- **Base Size**: 15px
- **Base Weight**: 600 (Semibold)

If the font files are not available, the system will fall back to system fonts.

## Converting Font Files:

If you have TTF files, you can convert them to WOFF2 using online tools like:
- https://cloudconvert.com/ttf-to-woff2
- https://convertio.co/ttf-woff2/
